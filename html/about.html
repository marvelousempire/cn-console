What Does CN Console Do?

The CN Console is the main dashboard for the Contribution Network. It lets you:

  • See all contributions (apps, tools, handbooks)
  • Manage consoles and cartridges
  • View protocols and documentation
  • Access AI features and integrations
  • Manage passwords and network settings

What Is a Passport?

A Passport is a JSON file that tells the system who this app is and how to load it. Think of it like an ID card for apps.

  • UDIN ‚Äî Unique ID (TYPE-NAME-TIMESTAMP)
  • Settings ‚Äî How settings are stored (storageKey prevents collisions)
  • Health ‚Äî Where to check if the app is running
  • Discovery ‚Äî Where to find cartridges to load
