# ai-console Cartridge

A Sunday App Framework cartridge for Quick Server.

## Installation

```bash
# Clone into your Quick Server cartridges folder
cd apps/quick-server/cartridges
git clone https://github.com/marvelousempire/qs-ai-console.git ai-console
```

## Usage

This cartridge is automatically discovered by Quick Server when placed in the `cartridges/` folder.

## Structure

```
ai-console/
├── index.html              # Main view
├── cartridge.settings.json # Settings with inheritance
├── cartridge.schema.json   # Schema validation
├── css/                    # Styles
└── js/                     # Scripts
```
